# Calculadora de consumo de energia
Essa calculadora foi criada utilizando a linguagem Python
essa calculadora tem como objetivo mostrar consumo mensal em kWh e custo mensal de aparelhos doméstivos.

#formula usada para calculo.

para o consumo foi utilizada a formula: 
(Watts * horas * dias) / 1000
(potencia * horas_dia * 30) / 1000

para o custo foi utilizada a formula:
consumo_mensal * 0.75 (0.75 foi o valor fixo utilizado como base).

Para executar o programa pelo vscode, clique em run python file e siga as etapas solicitadas
e/ou certifique se de ter o Python instalado.
